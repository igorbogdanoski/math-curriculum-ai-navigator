# Водич за интеграција на матурски тестови (ДИМ формат)

## Статус: Чека на PDF фајлови од МОН/ДИМ

---

## Кој формат на PDF е оптимален?

### ✅ Идеален (директна интеграција, без дополнителна обработка)

**Text-layer PDF** — PDF со вграден текст (не скенирана слика).

Препознавање: Ако можете да означите/копирате текст во PDF читачот → text-layer е присутен.

Повеќето официјални ДИМ документи се text-layer. Ова е случај и со МОН програмите.

### ⚠️ Прифатливо (потребна OCR предобработка)

**Скениран PDF** — слика скенирана во PDF.

Решение: Конвертирај со Adobe Acrobat / Gemini Files API / Google Document AI.

---

## Структура на PDF за максимална автоматизација

За да може Gemini да ги извлече прашањата **без рачна корекција**, PDF треба да следи:

### Структура на прашање

```
1. Наслов на прашањето (целосен текст)

   А) Одговор А
   Б) Одговор Б
   В) Одговор В
   Г) Одговор Г
```

### Клуч за одговори (на крај на документот или во посебен фајл)

```
Клуч за одговори:
1. В
2. А
3. Г
...
```

### Опционо (но корисно за AI контекст)

```
Тема: Тригонометрија
Ниво: Гимназиско XII
Година: 2024
Поени: 2
```

## Процес на интеграција (чекор по чекор)

### Чекор 1 — Испратете PDF

Испратете ги PDF фајловите на следниот начин (по приоритет):

1. **Директно во разговорот** — прикачете PDF и пишете: `Парсирај ги прашањата`
2. **Преку GitHub** — додадете во `docs/matura-pdfs/`
3. **Google Drive линк** — јавен линк

### Чекор 2 — Автоматска екстракција

Ќе го употребиме **Gemini Files API** за автоматска екстракција:

```
POST /api/parse-matura-pdf
→ Gemini 2.5 Flash чита PDF
→ Екстрахира прашања во JSON формат
→ Зачувува во data/matura/{year}.ts
```

### Чекор 3 — Резултат (JSON структура)

```json
{
  "year": 2024,
  "track": "gymnasium",
  "gradeLevel": 12,
  "questions": [
    {
      "id": "mat-2024-q1",
      "questionNumber": 1,
      "questionText": "Ако f(x) = sin(2x), колкав е f'(π/4)?",
      "choices": {
        "А": "0",
        "Б": "1",

        "Г": "-1"
      },
      "correctAnswer": "В",
      "topic": "Деривати",
```

## Тековно достапни PDF ресурси (јавни)

### МОН / ДИМ (Државна испитна матура)

- **Портал:** http://www.dim.edu.mk → Архива на тестови
- **Достапно:** Тестови 2015–2024 (математика)
- **Формат:** Text-layer PDF ✅

### МОН Наставни програми (за Н4 curriculum data)

- **Портал:** http://bro.gov.mk → Наставни програми → Средно образование
- **Достапно:** Програми по математика за сите типови
- **Формат:** Text-layer PDF ✅ — идеален за автоматска екстракција

## Препорака: Еден PDF = Едно одделение × Еден тип

```
mat_gymnasium_2024.pdf      ← сите прашања, Гимназиско, 2024
mat_vocational4_2024.pdf    ← Стручно 4-год, 2024
curriculum_gymnasium.pdf    ← МОН програма, Гимназиско X–XII
curriculum_vocational4.pdf  ← МОН програма, Стручно 4-год X–XII
curriculum_vocational3.pdf  ← МОН програма, Стручно 3-год X–XI
```

## Следни чекори по добивање на PDF

1. Испратете PDF → ќе парсираме со Gemini 2.5 Flash
