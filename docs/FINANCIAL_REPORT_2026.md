## 1. Моментална состојба

### 1.1 Производ и пазар
MisMath AI е прва македонска AI платформа за наставници по математика. Платформата:
- Генерира писмени работи, квизови, годишни и тематски планирања во < 2 минути
- Работи со МОН наставната програма, БРО стандардите и 45/40-минутната структура на часот
- Поддржува македонски, албански и турски јазик
- Достапна на: ai.mismath.net

### 1.2 Моментален ценовник
| План | Цена | Вредност |
|------|------|----------|
| Бесплатно | 0 МКД | 50 почетни AI кредити |
| Pro Наставник | 1 500 МКД / год. | Неограничени генерации, 12 месеци |
| Училишна лиценца | По договор | Сè од Pro + администрација |

**= 125 МКД / месечно = 2,05 EUR / месечно**

### 1.3 Состојба по вебинарот (1 јули 2026)
- **Присуство на вебинар:** 250+ наставници
- **Платформа активна:** јули 2026
- **Очекувана конверзија Free→Pro:** 10–20% (стандард за EdTech B2C)

## 2. Модел на трошоци

### 2.1 Инфраструктурни трошоци (месечно)
| Компонента | Провајдер | Трошок / мес. |
|-----------|-----------|--------------|
| Hosting | Vercel Pro | ~$20 (≈1 200 МКД) |
| База на податоци | Firebase Blaze | ~$15–40 (≈900–2 400 МКД) |
| AI генерации (Gemini API) | Google Cloud | ~$30–100 (≈1 800–6 000 МКД) |
| Домен + SSL | годишно | ≈600 МКД / год. |
| **Вкупно мин.** | | **≈4 500 МКД / мес.** |
| **Вкупно макс.** | | **≈10 200 МКД / мес.** |

> *AI трошокот расте со бројот на корисници — потребен е мониторинг.*

### 2.2 Трошок по корисник (проценка)
- Просечно генерирање: 3 AI повика / ден × 0,001 $ = ~0,10 $ / ден / активен корисник
- Месечен трошок по активен Pro корисник: ~$3 (≈185 МКД)
- **Приход по Pro корисник:** 125 МКД / месец
- **Маржа:** Негативна ако > 1 AI повик / 1,5 мин.

> ⚠️ **Клучен ризик:** Без лимит на Free корисниците, AI трошокот може да ги надмине приходите. Препорака: 50 вкупни кредити за Free (веќе имплементирано).

## 7. Буџет за следната фаза

### Препорачан буџет (авг–дек 2026): ~30 000 МКД
| Позиција | Сума |
|----------|------|
| Инфраструктура (5 мес.) | 22 500 МКД |
| Маркетинг (Facebook/Instagram наставници) | 5 000 МКД |
| Правна регулатива (регистрација на компанија) | 2 500 МКД |
| **Вкупно** | **30 000 МКД** |

### Break-even точка
- При 1 500 МКД/Pro корисник/год. = 125 МКД/мес.
- Мин. трошок: 4 500 МКД/мес.
- **Break-even: 36 Pro корисници** (достижно во авг–сеп 2026)
