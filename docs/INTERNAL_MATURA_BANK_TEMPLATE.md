## Структура на базенот — препорачано

За 120 прашања, препорачана распределба:

| Тип | Број | Поени |
|-----|------|-------|
| MC (1 поен, DoK 1–2) | 80 | 80 |
| Отворени (4–5 поени, DoK 2–4) | 40 | 160–200 |

По тема (за вок4-IT):
- Алгебра/функции: ~30 прашања
- Анализа (деривати + интеграли): ~25 прашања
- Матрици: ~15 прашања
- Статистика + веројатност: ~20 прашања
- Тригонометрија: ~10 прашања
- Броеви + комбинаторика: ~20 прашања

## Workflow за внес од PDF

1. **Отвори PDF** и темплејт JSON паралелно
2. За секое прашање, само пополни:
   - `questionText` (со LaTeX за формули — `$f(x) = x^2$`)
   - `choices` или `null`
   - `correctAnswer`
   - `topic` (слободен текст)
   - `topicArea` (од табелата погоре)
   - `conceptIds` (1–2 ID-а од табелата)
   - `points` (1 за MC, 4–5 за отворени)
3. `dokLevel`, `hints`, `aiSolution` — може да се пополнат подоцна
4. По завршување: `git add` + `git commit`

### LaTeX кратка референца
```
Полином:     $x^2 + 3x - 2$
Дропка:      $\frac{a}{b}$
Корен:       $\sqrt{x}$
Индекс:      $x_1, x_2$
Граница:     $\lim_{x \to 0}$
Интеграл:    $\int_0^1 f(x)\,dx$
Матрица:     $\begin{pmatrix} a & b \\ c & d \end{pmatrix}$
Стрелки:     $\Rightarrow$ (импликација), $\Leftrightarrow$ (еквиваленција)
```

## За кој track?

| Наставник | `track` поле | `conceptIds` префикс |
|-----------|-------------|----------------------|
| Стручно 4-год, ИТ | `"vocational-it"` | `voc4-*` |
| Стручно 4-год, Економија | `"vocational-economics"` | `voc4-*` |
| Гимназиско | `"gymnasium"` | `gym-*` |
| Стручно 3-год | `"vocational3"` | `voc3-*` |

> Ако базенот е универзален (сите стручни 4-год), користи `"vocational4"` и покасно може да се клонира по track.
