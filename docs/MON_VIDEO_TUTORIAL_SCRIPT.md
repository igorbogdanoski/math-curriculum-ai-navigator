**Јазик:** Македонски
**Алат за снимање:** OBS Studio (бесплатен) или Loom
**Резолуција:** 1920×1080, 30 fps, mic level −12 dB
**Distribution:** YouTube unlisted → embed во `MON_USER_GUIDE_MK.md`

## Подготовка пред снимањето

1. Логирај се на демо сметка: `teacher@mon-demo.ai.mismath.net` / `MonDemo!2026`
2. Отвори `?demo=mon` URL → потврди банер „МОН демо режим"
3. Затвори ги сите други табови (мрежна тишина за audio)
4. Активирај browser zoom 110% за читливост

## Сценарио по секунди

### 0:00 — 0:20 · Воведен слајд
- **Visual:** Логото на платформата + наслов „Math Curriculum AI Navigator — преглед за МОН"
- **Voice:** „Добредојдовте. Денес ќе ги покажам петте клучни процеси за наставник по математика на нашата платформа: годишен план, подготовка за час, материјали, матура и аналитика."

### 0:20 — 1:20 · Flow 1 — Годишен план
- **Action:** Наслов → `Annual Planner`
  1. Избери одделение (8.)
  2. Кликни „Генерирај годишен план"
  3. Покажи AI-генериран план со 8 единици → 36 часа
  4. Едитирај една единица (drag & drop)
- **Voice:** „AI генерира основен скелет според БРО стандардите. Наставникот го прилагодува, додава ресурси и зачувува."
- **Key callout:** „Усогласеност со БРО — секоја цел е мапирана на национален стандард"

### 1:20 — 2:30 · Flow 2 — Подготовка за час
- **Action:** Од планот → клик на единица → „Креирај подготовка"
  1. Auto-popup: тема, цели, активности
  2. Демо `AILessonAssistant` — постави прашање „Како да воведам разломки?"
  3. Покажи differentiation panel (поддршка / стандард / напредно)
  4. Експорт → Word
- **Voice:** „Подготовката се адаптира на нивото на класот. AI асистентот дава методички идеи во реално време. Експортот е компатибилен со постоечки училишни обрасци."

### 2:30 — 3:30 · Flow 3 — Материјали + квиз
- **Action:** Sidebar → `Materials Generator`
  1. Избери концепт (нпр. „Квадратни равенки")
  2. Тип = квиз, DoK = 1–4 mix, прашања = 8
  3. Click Generate → 4-5 sec
  4. Покажи дека секое прашање има solution + misconception note
  5. „Save to Library" → споделување со ученик
- **Voice:** „Платформата генерира педагошки балансирани материјали. Сите AI излези се валидирани со Zod schema — нема халуцинации во публикуваниот материјал."

### 3:30 — 4:20 · Flow 4 — Матура практика
- **Action:** Sidebar → `Matura Practice`
  1. Избери „Септември 2024 — основно ниво"
  2. Кликни „Стартувај тренинг"
  3. Реши 1 прашање → покажи instant feedback + IRT update
  4. Покажи `Matura Analytics` heatmap по концепт
- **Voice:** „Над 800 матурски прашања се внесени во базата. IRT моделот (3-параметриски логистички) го оценува нивото на ученикот по секое прашање."

### 4:20 — 4:50 · Flow 5 — Аналитика
- **Action:** Sidebar → `Teacher Analytics`
  1. Преглед на класот: просечен score, distribution
  2. Drill-down на ученик X → misconception list + препораки
  3. Покажи export → PDF report за родител
- **Voice:** „Наставникот добива конкретни препораки кој ученик треба засилена работа на која подкомпонента."

### 4:50 — 5:00 · Завршен слајд
- **Visual:** Логото + контакт `info@ai.mismath.net` + статус-страна `status.ai.mismath.net`
- **Voice:** „Платформата е спремна за пилот во избрани училишта. Контакт за дополнителни прашања е во описот. Ви благодариме."
