# План за Развој: Math Curriculum AI Navigator 🚀

Овој документ служи како стратегија за имплементација на новите функционалности, поделени по фази и приоритет.

## 🏁 Фаза 1: "Наставник 2.0" (Фокус: Продуктивност и Администрација)
*Оваа фаза решава конкретни административни проблеми на наставниците денес.*

### 🛠️ Phase 0: Технички Долг (Technical Debt) ✅ ПРИОРИТЕТ
**Статус:** ЗАВРШЕНО
- [x] **Кеширање:** React Query / `setCached` во Firestore за да не се трошат AI кредити за исти генерирања.
- [x] **Error Boundary:** Подобрено справување со грешки (Graceful degradation).
- [x] **Env Vars:** Проверка на API клучеви и конфигурација (особено за Paid Tier квоти).


### 1.1. "Killer Feature": Генератор на Тестови (Група А / Група Б) ⭐ ПРИОРИТЕТ
**Цел:** Автоматско генерирање на PDF тестови со две групи и лист со решенија.
- [x] **UI:** Интерфејс за избор на тема, број на задачи, и тежина.
- [x] **AI Логика:** Промпт инженерство за генерирање „паралелни“ задачи (исти тип, различни бројки) за Група А и Б.
- [x] **PDF Генерација:** Користење на `react-pdf` или `jspdf` за форматирање на тестовите спремни за печатење.
- [x] **Решенија:** Генерирање на посебна страница само со точните одговори за наставникот.

### 1.2. Напреден Експорт (Word/PDF)
**Цел:** Документите да изгледаат официјално (како за БРО/Инспекторат).
- [x] **Word Експорт:** Имплементација на документ со заглавје, табели и официјален фонт.
- [x] **Темплејти:** Можност наставникот да си сними свој „заглавје“ (Име на училиште, Наставник).

### 1.3. "Magic Edit" (AI Ревизија)
**Цел:** Брза промена на веќе генериран содржина.
- [x] **UI:** Селектирање на текст -> копче "Подобри со AI".
- [x] **Опции:** "Поедностави", "Скрати", "Направи поинтересно", "Прилагоди за ученици со попреченост".

## 🎮 Фаза 2: "Интерактивна Училница" (Фокус: Ученици)
*Вклучување на учениците преку нивните телефони/таблети.*

### 2.1. Квизови во Живо (QR Code)
**Цел:** Брза проверка на знаење на час (Формативно оценување).
- [x] **Host Mode:** Наставникот пушта квиз на проектор (додаден QR код во LiveTab).
- [x] **Student Mode:** Учениците скенираат QR и одговараат (без логирање, само име) на `/live`.
- [x] **Real-time:** Резултатите се појавуваат на табла во реално време (користејќи Firestore).

### 2.2. AI Тутор за Ученици (Safe Chat)
**Цел:** Помош при учење, без "пишување на домашната".
- [x] **System Prompt:** Строги правила за AI (објаснувај, не решавај).
- [x] **Контекст:** Ботот знае која лекција се учи моментално.

## 🚀 Фаза 4: "Ентерпрајз & Next-Gen AI" (Фокус: Комерцијализација и Институционализација)
*Оваа фаза ја издигнува платформата на светско SaaS ниво, подготвена за национална имплементација.*

### 4.1. Автоматско Оценување преку Камера (AI Vision Grading) 📷
- [ ] **Скенирање:** PWA модул за сликање на физички тестови.
- [ ] **AI Анализа:** Користење на Gemini Vision за читање ракопис и чекор-по-чекор барање на грешки во математичките процеси.
- [ ] **Интеграција:** Автоматско префрлање на резултатите во базата и дашбордот.

### 4.2. Адаптивно Учење (Adaptive Learning - Домашни задачи) 🎯
- [ ] **Персонилизирани PDF-и:** Автоматско генерирање на различни домашни задачи базирани на резултатите од квизот (послабите добиваат визуелни помагала, напредните посложени проблеми).
- [ ] **Автоматска дистрибуција:** Доделување на задачите на индивидуални ученички профили.

### 4.3. RAG Интеграција (Официјални Учебници и БРО) 📚
- [ ] **Векторска База:** Поставување на Pinecone или Firebase Vector Search.
- [ ] **Внесување Податоци:** Трансформирање на сите наставни програми (од 1-во до 9-то одд., потоа средно) и достапни учебници во векторски формат.
- [ ] **Прецизно Цитирање:** Менување на промптовите за AI-то секогаш да мора да се базира *исклучиво* на официјалните материјали кога креира лекции.

### 4.4. Институционален Дашборд (Multi-tenant SaaS) 🏢
- [ ] **Улоги:** Систем за автентикација базиран на улоги (Ученик, Наставник, Родител, Директор/Инспектор).
- [ ] **Макро-Аналитика:** Поглед за директорот со "хеликоптерска" перспектива врз учинокот на целото училиште (по паралелки и наставници).
- [ ] **Родителски Портал:** Едноставен графички интерфејс каде родителите добиваат AI извештаи за прогресот на детето.
